## Objetivo

Implementar sistema de importação de catálogos de fabricantes de alumínio (Hydro, Alumasa, etc.) usando IA (Claude API) para extrair automaticamente perfis e modelos de janelas/portas a partir de PDFs ou texto.

## Acceptance Criteria

- [x] AC1: Tabela `ai_import_jobs` para rastreamento de jobs de importação
- [x] AC2: CatalogImportService extrai perfis e modelos via Claude Haiku API
- [x] AC3: Pipeline assíncrono: upload → processamento IA → review → confirmação
- [x] AC4: CatalogImportDialog com UX completa (4 steps: upload, processing, review, done)
- [x] AC5: Validação dos dados extraídos antes de salvar no catálogo
- [x] AC6: Upsert idempotente em `perfis_aluminio` e `window_models` (onConflict: company_id,codigo)
- [x] AC7: Evento `catalog.import.completed` emitido ao concluir

## Arquitetura

```
Upload PDF/texto
  → CatalogImportService.iniciarImportacao()
  → Cria ai_import_jobs (status: processando)
  → processarJob() [async — fire and forget]
    → extrairComIA() [Claude Haiku API]
    → validarDadosExtraidos()
    → Salva ai_raw_output + dados_para_import
    → status: concluido
  → UI poll até concluir
  → Review: fabricante, totalPerfis, totalModelos, confiança, avisos
  → confirmarImportacao()
    → Upsert em perfis_aluminio
    → Upsert em window_models
  → catalog.import.completed emitido
```
