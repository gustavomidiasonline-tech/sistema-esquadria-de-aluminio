## Objetivo

Criar a estrutura de banco de dados para a biblioteca tecnica de modelos de esquadrias, incluindo as pecas (perfis) necessarias para cada modelo com formulas de calculo de comprimento.

## Contexto

Atualmente o sistema tem a tabela `produtos` (tipo de esquadria) e `perfis_aluminio` (perfis por produto). Porem, faltam:
- Modelos de esquadria com parametros de dimensionamento
- Formulas de calculo de comprimento de cada peca (ex: "largura - 2*folga")
- Biblioteca de vidros, ferragens e acessorios com SKU e preco

Essa estrutura e a base para o motor de calculo automatico.

## Schema

```sql
-- Modelos de esquadrias
CREATE TABLE window_models (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  company_id UUID REFERENCES companies(id),
  codigo TEXT NOT NULL,
  nome TEXT NOT NULL,
  tipo TEXT NOT NULL,  -- correr, balcao, maxim-ar, fixo, etc
  num_folhas INTEGER DEFAULT 1,
  descricao TEXT,
  largura_min NUMERIC,
  largura_max NUMERIC,
  altura_min NUMERIC,
  altura_max NUMERIC,
  ativo BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Pecas de cada modelo (perfis necessarios)
CREATE TABLE window_parts (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  window_model_id UUID NOT NULL REFERENCES window_models(id),
  perfil_aluminio_id UUID REFERENCES perfis_aluminio(id),
  posicao TEXT NOT NULL,        -- ex: 'marco_horizontal', 'folha_vertical'
  formula_comprimento TEXT NOT NULL, -- ex: 'largura - 40', 'altura / num_folhas'
  quantidade INTEGER NOT NULL DEFAULT 1,
  observacao TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Tipos de vidro
CREATE TABLE glass_types (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  company_id UUID REFERENCES companies(id),
  codigo TEXT NOT NULL,
  nome TEXT NOT NULL,
  espessura_mm NUMERIC NOT NULL,
  tipo TEXT,  -- temperado, laminado, comum, insulado
  preco_m2 NUMERIC NOT NULL DEFAULT 0,
  fornecedor TEXT,
  ativo BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Ferragens
CREATE TABLE hardware (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  company_id UUID REFERENCES companies(id),
  codigo TEXT NOT NULL,
  nome TEXT NOT NULL,
  tipo TEXT,  -- fechadura, dobradica, puxador, trilho
  unidade TEXT DEFAULT 'un',
  preco_unitario NUMERIC NOT NULL DEFAULT 0,
  fornecedor TEXT,
  ativo BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Acessorios
CREATE TABLE accessories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  company_id UUID REFERENCES companies(id),
  codigo TEXT NOT NULL,
  nome TEXT NOT NULL,
  tipo TEXT,
  unidade TEXT DEFAULT 'un',
  preco_unitario NUMERIC NOT NULL DEFAULT 0,
  fornecedor TEXT,
  ativo BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);
```
