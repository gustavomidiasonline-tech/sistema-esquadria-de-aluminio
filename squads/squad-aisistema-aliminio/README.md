# squad-aisistema-aliminio

Sistema completo ERP que compila vários sistemas do mercado de esquadrias de aluminio e vidraçarias em um só, com plano de corte, CRM, estoque de peças, catalogos de produtos, controle financeiro, nota fiscal, dashboard em tempo real, orçamentos, check, tipologias de esquadrias

## Installation

This squad is installed locally in your project:

```
./squads/squad-aisistema-aliminio/
```

## Usage

Activate agents from this squad and use their commands.

### Available Agents

- **example-agent** - Example agent (customize or remove)

### Available Tasks

- **example-agent-task** - Example task (customize or remove)

## Configuration

This squad extends the core AIOS configuration.

## Development

1. Add agents in `agents/` directory
2. Add tasks in `tasks/` directory (task-first architecture!)
3. Update `squad.yaml` components section
4. Validate: `@squad-creator *validate-squad squad-aisistema-aliminio`
